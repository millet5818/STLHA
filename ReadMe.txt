English:
1. Configure environment variables
    1).  Add the directory containing gdal3.2.0_bin in the project directory (i.e.,.../FSModel/libs/gdal3.2.0_bin) to the system environment variable Path. 
    2).  Create a new system environment variable named PROJ_LIB, with the value being the directory of libs (i.e.,.../FSModel/libs)

Chinese:
1. 配置环境变量(2021-01-01)
	1). 将项目目录下的gdal3.2.0_bin所在目录(比如:.../FSModel/libs/gdal3.2.0_bin)，
		添加到系统环境变量Path中。
	2). 新建一个系统环境变量命名为PROJ_LIB，变量值为libs所在目录（比如:.../FSModel/libs）。
